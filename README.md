# 教科書シェア v13

学生同士で不要になった教科書を、金銭の仲介なしで譲渡する流れを想定した3層構造のWebアプリです。公開サイトには、架空の利用者・教科書・教科・教員・会話だけを登録します。

## 公開デモ上の注意

- 掲載情報はすべて授業課題用の架空データです。
- 実在する個人名、授業情報、メールアドレスなどは登録しないでください。
- ログイン方式は授業用の簡易方式であり、本番サービス向けの認証ではありません。

## v13の主な改善

- 教員名を `教員A01`、著者名を `著者A01` の形式へ統一
- 利用者名を `利用者A01` の形式へ統一
- 装飾的な科目名を廃止し、一般的な大学科目名へ変更
- 8つの学問分野に教科・教科書・出品・履修・関連データを均等配置
- 旧版の装飾的な接頭辞や、データ名に含まれる「デモ」を検出するテストを追加
- RenderのExternal Database URLを入力して公開DBを安全に置換するスクリプトを追加
- PWAキャッシュをv13へ更新

## 架空データの分野構成

次の8分野を使用します。

1. 文学・語学
2. 経済・経営
3. 法学・政治
4. 理工・情報
5. 自然科学
6. 医療・生命科学
7. 社会・教育
8. 政策・環境

各分野には、教科5件、教科書5冊、利用者3人、出品10件を登録します。

教科名の例：

- 日本文学概論
- ミクロ経済学
- 憲法
- 数学Ⅰ
- アルゴリズム理論
- 物理学Ⅰ
- 解剖学
- 社会学
- 環境科学

## データ件数

`npm run db:reset-demo` は、接続先DBの既存データをすべて削除した後、次のデータを投入します。

- アカウント：24件
- 教科書：40冊
- 教科：40件
- 出品：80件
- チャット：80件
- 履修登録：120件
- お気に入り：192件
- 閲覧履歴：192件
- 保存検索：24件
- 推薦フィードバック：16件
- 通知：24件
- 既読状態：24件

## サンプルアカウント

パスワードはありません。ログイン画面のボタンから入力できます。

```text
出品者として試す
利用者A02 / webpro-sample-a02@keio.jp

受取希望者として試す
利用者A01 / webpro-sample-a01@keio.jp
```

## 主な機能

- 名前・大学メールアドレスによる簡易ログインとアカウント作成
- 教科書の出品、編集、取り消し、譲渡完了
- 教科書名・教科名・著者名・担当教員名による検索
- 本の状態、取引状態、並び順による絞り込み
- 譲渡品ごとのチャットと交渉相手の選択・変更
- お気に入り、閲覧履歴、保存検索、通知
- 履修科目・お気に入り・閲覧履歴を使ったおすすめ
- PWA、スマートフォン表示、キーボード操作への対応

## 3層構造

- **フロントエンド**：EJS、HTML、CSS、JavaScript
- **バックエンド**：TypeScript、Express
- **データベース**：PostgreSQL、Prisma

## 通常の起動

```bash
npm install
npm run db:generate
npm run db:push
npm test
npm start
```

## Render DBをv13の架空データへ置換

Render DashboardのPostgreSQL画面からExternal Database URLを取得し、次を実行します。

```bash
bash scripts/reset-render-demo-v13.sh
```

スクリプトは接続先ホストとDB名を表示し、`RESET` の入力後にだけ既存データを削除します。DB初期化コマンドはRenderのBuild Commandへ入れないでください。

## Render設定例

```text
Build Command:
npm clean-install && npx prisma generate && npx prisma db push

Start Command:
npm start
```
